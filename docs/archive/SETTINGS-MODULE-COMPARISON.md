### 2. RLS Policies Too Permissive

**Status:** ⚠️ **SECURITY CONCERN**
**Issue:** All authenticated users can modify all settings
**Current:**
```sql
CREATE POLICY "Authenticated users can update settings"
  ON app_settings FOR UPDATE TO authenticated
  USING (true) WITH CHECK (true);
```
**Impact:** MEDIUM - Any authenticated user can change system settings
**Recommendation:** Restrict to admin role only:
```sql
CREATE POLICY "Only admins can update settings"
  ON app_settings FOR UPDATE TO authenticated
  USING (
    EXISTS (
      SELECT 1 FROM user_profiles
      WHERE user_profiles.id = auth.uid()
      AND user_profiles.role = 'admin'
    )
  );
```
